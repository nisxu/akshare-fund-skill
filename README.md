# akshare-fund-skill

基于 AKShare 的智能基金筛选与管理 OpenClaw Skill。

## 功能模块

1. **基金数据查询与筛选** - 多维度条件筛选（类型、规模、收益率、夏普比率、最大回撤等）
2. **基金详情深度分析** - 净值走势、持仓明细、基金经理信息
3. **用户持仓管理** - 持仓记录、盈亏跟踪、数据持久化
4. **持仓分析与诊断** - 资产配置分析、风险评估、健康度评分
5. **智能调仓建议** - 止盈止损、再平衡、优化替换建议
6. **市场行情研判** - 宏观数据跟踪、估值分位监控、定期操作建议

## 安装

```bash
pip install -r requirements.txt
```

## 数据源

- [AKShare](https://github.com/akfamily/akshare) - 开源财经数据接口库
